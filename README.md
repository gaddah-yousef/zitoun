# 🍽️ ZITOUN - Restaurant Website

## 📌 Description
ZITOUN est un site web moderne pour un restaurant, développé en HTML et CSS.  
Il présente le menu, les informations de contact et une interface utilisateur simple et élégante.

---

## 🚀 Fonctionnalités

- Design responsive (mobile + desktop)
- Navigation simple
- Pages :
  - Accueil
  - Menu
  - Contact

---

## 🛠️ Technologies

- HTML5
- CSS3

---

## 📂 Structure

restaurant/
- index.html
- menu.html
- contact.html
- css/
- images/
